# Sophium: Agentic AI Personal CFO
**Google Agent Labs Hackathon 2026**

Sophium is an Agentic AI Personal CFO designed to help users manage their financial behaviors, simulate scenario-based financial decisions, track target milestones, and receive explainable recommendations. 

Instead of a generic chatbot, Sophium implements a modular **Agent-to-Agent (A2A) collaborative workflow** using the **Google Agent Development Kit (ADK)** and stores long-term memory in **Qdrant**.

---

## 🚀 Key Features

*   **A2A Orchestration**: Pipelines complex queries sequentially from RAG knowledge lookups down to compound math projections and timeline reviews.
*   **Observability Timeline**: Floating sidebar tracer displaying agent runtimes, vector database searches, and execution details for judges.
*   **Multi-Layer Memory (Qdrant)**: Stores semantic preferences, active goal parameters, and recent conversation turns in specialized payloads.
*   **Domain Knowledge (RAG)**: Separates user profile memories from external financial constraints (e.g. Section 80C tax rules and historical index fund yields).
*   **Explainable AI (XAI)**: Enforces structured Pydantic formatting containing confidence scores, risks, rationale details, and alternative suggestions.
*   **One-Click Hackathon Demo Mode**: Seeds the SQLite database, vector memory collections, and logs you into a sandbox workspace instantly.

---

## 🛠️ Tech Stack

*   **Backend**: Python, FastAPI, SQLModel (SQLite), Google Generative AI (Gemini Pro)
*   **Agent Framework**: Google Agent Development Kit (ADK)
*   **Vector Database**: Qdrant (local Docker or Cloud)
*   **Frontend**: React + Vite, TypeScript, Zustand, Recharts, Tailwind CSS (v4)

## ⚡ Setup & Launch Instructions

### 1. Run Qdrant Vector DB
Start the vector store container:
```bash
docker compose up -d
```
*(Sophium includes an automatic in-memory fallback. If Docker is not running, the system will fall back to local storage automatically.)*

### 2. Configure & Start the Backend
1. Create a `.env` file in the `backend/` directory:
   ```env
   PORT=8000
   DATABASE_URL=sqlite:///./sophium.db
   JWT_SECRET_KEY=supersecretkey_changeinproduction_forhackathon2026
   ACCESS_TOKEN_EXPIRE_MINUTES=1440
   GEMINI_API_KEY=your_gemini_api_key_here
   QDRANT_URL=http://localhost:6333
   ```
2. Initialize virtual environment and start:
   ```bash
   cd backend
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt
   uvicorn app.main:app --reload
   ```

### 3. Configure & Start the Frontend
1. Start the Vite client dev server:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
2. Open the browser at [http://localhost:5173](http://localhost:5173).

## 📽️ The 2-Minute Demo Journey

Follow this sequence to showcase Sophium during judging:

1.  **Demo Access**: Click **Launch 2-Min Demo Mode** in the top banner. This populates SQLite, writes memories into Qdrant, and signs you in as Rohan.
2.  **Inspect Charts**: Verify categorized allocations and historical trends on the dashboard.
3.  **Run Simulation**: Ask *"I want to buy a house in five years"* in the CFO console.
4.  **Trace Telemetry**: Check the right-hand **Agent Activity Trace** to inspect the sequential ADK execution details and runtimes.
5.  **Examine Explanations**: Verify the sub-cards (Confidence, rationale, risks, alternatives) generated by the Explanation Agent.
6.  **Proactive alerts**: Observe the dashboard banner alerts when transaction trends compromise target milestones.
